## 🎯 测试环境准备

### 1. 前端服务

```bash
cd data-share-demo/frontend
npm install
npm run dev
```

访问：`http://localhost:5173`

### 2. 后端服务（三选一）

#### 选项A：Mock Backend（快速测试）
```bash
cd data-share-demo
python mock_backend.py
```
- ✅ 优点：无需数据库，快速启动
- ❌ 缺点：数据是假的

#### 选项B：真实 Backend（完整测试）
```bash
cd data-share-demo/backend
npm install
npm start
```
- ✅ 优点：真实数据，完整功能
- ❌ 缺点：需要配置数据库

#### 选项C：在线 Backend（远程测试）
修改 `src/api.js`：
```javascript
export const api = axios.create({
  baseURL: "https://your-backend-url.com",
});
```

## 🔌 接口对接测试

### 测试准备

1. **打开浏览器开发者工具**（F12）
2. **切换到 Network 标签**
3. **执行测试操作**
4. **查看网络请求**

---

### 接口测试检查清单

#### 1. 登录接口

**操作：** 登录

**检查项：**
- [ ] 请求方法：`POST`
- [ ] 请求 URL：`/api/auth/login`
- [ ] 请求头：`Content-Type: application/json`
- [ ] 请求体：包含 username 和 password
- [ ] 响应状态：`200`
- [ ] 响应体：包含 token 和 user 对象

**失败场景：**
- [ ] 错误密码返回 `401`
- [ ] 缺少字段返回 `400`

---

#### 2. 数据市场接口

**操作：** 访问数据市场

**检查项：**
- [ ] 请求方法：`GET`
- [ ] 请求 URL：`/api/datasets/market`
- [ ] 请求头：无需 Authorization
- [ ] 查询参数：domain, sort（可选）
- [ ] 响应状态：`200`
- [ ] 响应体：包含 list 数组

---

#### 3. 文件上传接口

**操作：** 上传数据集

**检查项：**
- [ ] 请求方法：`POST`
- [ ] 请求 URL：`/api/datasets`
- [ ] 请求头：`Authorization: Bearer <token>`
- [ ] Content-Type：`multipart/form-data`
- [ ] 请求体：FormData（name, domain, dataType, description, file）
- [ ] 响应状态：`201`
- [ ] 响应体：包含新创建的 dataset 对象

---

#### 4. 我的数据接口

**操作：** 访问我的数据页面

**检查项：**
- [ ] 请求方法：`GET`
- [ ] 请求 URL：`/api/datasets/mine`
- [ ] 请求头：`Authorization: Bearer <token>`
- [ ] 响应状态：`200`
- [ ] 响应体：包含 owned 数组

**未登录场景：**
- [ ] 返回 `401` 或跳转到登录页

---

### 常见接口问题

| 问题 | 可能原因 | 解决方案 |
|------|---------|---------|
| CORS 错误 | 后端未配置 CORS | 后端添加 CORS 中间件 |
| 401 错误 | Token 无效/过期 | 重新登录获取新 Token |
| 404 错误 | 接口路径错误 | 检查 baseURL 和端点路径 |
| 网络错误 | 后端未启动 | 确认后端服务运行 |
| 请求超时 | 文件过大 | 检查网络或文件大小 |

## 🐛 如果页面切换不工作

### 问题1：点击按钮没反应

**检查步骤：**

1. **打开浏览器开发者工具（F12）**
   - 切换到 Console 标签
   - 点击导航按钮时，查看是否有红色错误信息

2. **检查网络请求**
   - F12 > Network 标签
   - 点击导航按钮
   - 查看是否有失败的请求（红色）

3. **检查 Mock Backend**
   ```bash
   # 应该在终端看到类似输出：
   INFO: 127.0.0.1:xxxxx - "GET /api/datasets/mine HTTP/1.1" 200 OK
   ```

---

### 问题2：页面一直加载中

**原因：** 前端在等待后端响应

**解决：**

1. 检查 Mock Backend 是否运行：
   ```bash
   # 在浏览器访问
   http://localhost:3001
   
   # 应该看到：
   {"service": "数据市场 Mock Backend", "status": "running", ...}
   ```

2. 如果 Mock Backend 没运行，重新启动：
   ```bash
   cd D:\litonglab\deepanalyze\data-share-demo
   python mock_backend.py
   ```

---

### 问题3：看不到导航栏

**原因：** 可能未正确登录，或登录信息丢失

**解决：**

1. 检查登录状态（F12 > Console）：
   ```javascript
   console.log('Token:', localStorage.getItem('token'));
   console.log('User:', localStorage.getItem('user'));
   ```

2. 如果输出是 `null`，说明未登录，需要重新登录

3. 如果已登录但看不到导航栏，尝试：
   - 刷新页面（Ctrl+R）
   - 清除缓存后刷新（Ctrl+Shift+R）

## 💡 额外提示

### 快捷键
- `F5` - 刷新页面
- `Ctrl+Shift+R` - 强制刷新（清除缓存）
- `F12` - 打开开发者工具
- `Alt+←` - 浏览器后退
- `Alt+→` - 浏览器前进

### 浏览器前进后退
- 点击浏览器的 ← → 按钮也可以在页面间切换
- 每次切换页面，浏览器历史记录都会更新
